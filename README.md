# 🏦 Bank Management System (Python)

## Overview

Bank Management System is a Python-based console application that simulates basic banking operations. The project allows users to create bank accounts, deposit and withdraw money, update account details, view account information, and delete accounts.

All account data is stored in a JSON file, providing simple data persistence without requiring a database.

This project was built as part of my Python learning journey to strengthen my understanding of Object-Oriented Programming, file handling, and data management.

---

## Features

✅ Create a new bank account

✅ Generate a unique account number automatically

✅ Deposit money into an account

✅ Withdraw money from an account

✅ Update account details

✅ Display account information

✅ Delete an account

✅ Store account data using JSON
